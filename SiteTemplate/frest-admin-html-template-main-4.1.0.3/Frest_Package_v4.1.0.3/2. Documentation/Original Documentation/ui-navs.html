Navs

Documentation and examples for how to use Bootstrap’s included navigation components.
Read the official Bootstrap Documentation for a full list of instructions and other options.

Base Nav

Bootstrap includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.

The base .nav component is built with flexbox and provide a strong foundation for building all types of navigation components. It includes some style overrides (for working with lists), some link padding for larger hit areas, and basic disabled styling.

  • Preview
  • HTML
  • Active
  • Link
  • Link
  • Disabled 
<ul class="nav">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Disabled</a>
  </li>
</ul>

Available Styles

Change the style of .navs component with modifiers and utilities. Mix and match as needed, or build your own.

Horizontal Alignment

Fill and justify

Force your .nav’s contents to extend the full available width one of two modifier classes. To proportionately fill all available space with your .nav-items, use .nav-fill. Notice that all horizontal space is occupied, but not every nav item has the same width.

  • Preview
  • HTML 
    • <nav class="nav nav-pills nav-fill">
  <li class="nav-item">
    <button class="nav-link active" data-bs-target="javascript:void(0)">Active</button>
  </li>
  <li class="nav-item">
    <button class="nav-link" data-bs-target="javascript:void(0)">Much longer nav link</button>
  </li>
  <li class="nav-item">
    <button class="nav-link" data-bs-target="javascript:void(0)">Link</button>
  </li>
  <li class="nav-item">
    <button class="nav-link disabled" data-bs-target="javascript:void(0)" tabindex="-1" aria-disabled="true">Disabled</button>
  </li>
</nav>

    Working with flex utilities

    If you need responsive nav variations, consider using a series of flexbox utilities. While more verbose, these utilities offer greater customization across responsive breakpoints. In the example below, our nav will be stacked on the lowest breakpoint, then adapt to a horizontal layout that fills the available width starting from the small breakpoint.

    • Preview
    • HTML 
    <nav class="nav nav-pills flex-sm-nowrap flex-column flex-sm-row">
  <button class="flex-sm-fill text-sm-center nav-link active" data-bs-target="javascript:void(0)">Active</button>
  <button class="flex-sm-fill text-sm-center nav-link" data-bs-target="javascript:void(0)">Longer nav link</button>
  <button class="flex-sm-fill text-sm-center nav-link" data-bs-target="javascript:void(0)">Link</button>
  <button class="flex-sm-fill text-sm-center nav-link disabled" data-bs-target="javascript:void(0)" tabindex="-1" aria-disabled="true">Disabled</button>
</nav>
